This Jupyter Notebook performs data analysis on an Excel dataset using Python. The notebook demonstrates essential data analysis steps including data loading, inspection, statistical summaries, and group-based aggregation to extract meaningful insights from the dataset.

What this notebook includes:

Installing and importing required libraries (Pandas, NumPy)

Loading an Excel file into a Pandas DataFrame

Displaying and inspecting raw data

Generating descriptive statistics using describe()

Checking dataset dimensions with shape

Grouping data using groupby() for analytical insights

Aggregating numerical features such as Annual Salary and Age

Analyzing data by Job Title, Full Name, and Ethnicity

Tools & Technologies:

Python

Jupyter Notebook

Pandas

NumPy

Excel (.xlsx)

Practice Excel-to-Python data analysis

Learn Pandas operations like grouping and aggregation

Suitable for beginners in data analysis and data science
